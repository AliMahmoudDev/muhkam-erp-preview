# MUHKAM ERP — Apple-style Landing Preview

This is a **preview-only** deployment of the MUHKAM ERP landing page
with the new Apple-inspired redesign.

## What's included
- ✅ Landing page (Hero, Features, Pricing, FAQ, Footer) — fully redesigned
- ✅ Login page UI (visual only, no actual auth)
- ❌ Backend API is NOT included (this is a static preview)

## To deploy on Vercel
1. Import this repo on https://vercel.com/new
2. Vercel will auto-detect Vite framework
3. Click "Deploy" — that's it!

The landing page will be available at `https://muhkam-erp-preview.vercel.app/`
