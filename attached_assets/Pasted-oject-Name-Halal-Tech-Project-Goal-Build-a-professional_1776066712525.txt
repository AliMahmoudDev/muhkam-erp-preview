oject Name: Halal Tech (حلال تك) Project Goal: Build a professional B2B E-commerce Marketplace for Apple Spare Parts tailored for the Egyptian market. The app branding should reflect trust and integrity.

Language & Localization (Critical):

Primary Language: Arabic (AR).
Direction: RTL (Right-to-Left) layout for all pages.
Font: Use a professional Arabic Google Font like 'Cairo' or 'Tajawal' for a modern look.
Core User Flow & Authentication:

Registration Page: Create a detailed sign-up form including: Full Name, Mobile Number, Shop Name, Email, and Role (Dropdown: "فني صيانة" or "تاجر جملة").
Smart Address System: Implement a hierarchical dropdown system for Egypt: First dropdown for "المحافظات" (Governorates), and once selected, a second dropdown appears with "المدن" (Cities) belonging to that governorate.
Approval System: All new accounts must be set to status: "pending" by default. Users cannot access the store until an Admin changes their status to approved.
Quick Access: Add a floating WhatsApp button on the registration/login screen labeled "تفعيل سريع عبر واتساب".
The Storefront (User Experience):

Branding: Display the logo "Halal Tech" prominently in the header.
Dynamic Ad Banner: A high-quality image slider at the top of the home page. The Admin must be able to upload/change these banners.
Product Architecture: Organize products into categories (آيفون, آيباد, ساعة أبل) and sub-categories (شاشات, بطاريات, سوكيتات شحن, etc.).
B2B Pricing Logic: Implement role-based pricing. The product page should display different prices based on the user's role (سعر الفني vs سعر التاجر).
Product Page: Include high-res images, quality labels (أصلي, هاي كوبي, درجة أولى), and stock availability.
Admin Dashboard (The Control Center):

User Management: A table to view all pending users, approve/reject them, and change their roles.
Inventory Management: CRUD operations for products (Add, Edit, Delete, Update Stock).
Ad Manager: Interface to upload and manage the home page banners.
Order Tracking: A system to manage orders from "قيد المراجعة" to "تم الشحن" and "تم التسليم".
Technical Stack Preferences:

Frontend: React with Tailwind CSS for a modern, responsive design. Ensure full RTL support.
Backend: Node.js with a robust database (PostgreSQL/Prisma).
UI Style: Minimalist, Clean, Trustworthy (White, Black, and a accent color like Deep Green or Blue to match the "Halal" trust vibe).
Please start by setting up the database schema with Arabic support, then build the Authentication flow, followed by the Storefront and Admin Panel.
